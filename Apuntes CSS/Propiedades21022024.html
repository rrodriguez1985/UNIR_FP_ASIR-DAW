Propiedades CSS: Colores

Para poder estalecer colores en CSS, podemos hacerlo de la siguiente manera:
  • Color: podemos establacer el color de texto de nuestra web.
  • background-color: nos permite establecer el color de fondo de nuestra web o elementos

Para poder poner un color en un elemento CSS, podemos hacerlo de las siguientes maneras:
  • Palabra clave: Eje: red, blue, green, yellow, orange...
  • RGB (red, green, blue): Eje: rgb(46,58,205)
  • RGB con capa alpha: La capa alpha establece la tonalidad más clara o más oscura. Tiene valores de entre
    0 (más clara) o 1 (como más oscura). Eje: rgba(25,40,90,0.5)
  • Hexadecimal:Valores que van del 0-9 y A-F. Ej:#FF5628
  • Hexadecimal abreviado:Tomamos como valor prioritario el primer digito: Eje: #FF5628 --> #F52

Unidades en CSS

Dentro de las unidades de medida que podemos usar en CSS, podemos destacar dos grupos:
  • Unidades Absolutas:son un tipo de medida fija (no cambian nunca). Son ideales para medios que no son variables,
    puesto que son poco flexibles a la hora de usarlos. P.E: pulgadas, cm, picas, mm, pt.
  • Unidades Relativas:son un tipo de medida más común en CSS, puesto que depende de otra resolución y tienen una
    curva de aprendizaje aunque más compleja, son ideales para diferentes tamaños y para hacer webs responsive.
    em (1em=16px), pixel, %, rem, etc.
  • width: determinar el tamaño de anchura de los elementos
  • height: determinar el tamaño de altura de los elementos

Modelos de Cajas en CSS

Todos los elementos en HTML, cuando creamos nuestras páginas web, están dentro de lo que se conoce como cajas (div)
Las cajas que vamos a utilizar serán principalmente como se ha comentado div, y también podemos usar elementos o
etiquetas semánticas ya predefinidas: nav, aside, footer, header, article, etc.
A partir de ahora, en CSS cuando vayamos a maquetar, dejaremos de usar tablas por empezar a usar este tipo de cajas.
El motivo principal, es que las tablas no son elemento muy flexibles para maquetar en HTML, y por lo tanto, nos limitan
el que podamos adaptar nuestras web a distintos tipos de dispositivos.
Las Propiedades que podemos aplicar a las cajas y que tenemos que tener en cuenta son las siguientes:
  • Margin: Corresponde al margen existente entre el borde exterior de nuestra caja / elemento contenedor y nuestro navegador
  • Padding: Corresponde al margen interior de nuestra caja, es decir, al espacio existente entre el contenido y el borde
    de nuestro elemento contenedor
  • Contenido: Son las etiquetas que van a ir dentro de nuestra caja o elemento contenedor.
