Definición de CLASS en CSS

En HTML y CSS, disponemos de una serie de atributos que nos permiten seleccionar ciertos elementos o etiquetas para asignarles
una serie de características concretas con respecto al resto de elementos que forman nuestra web.
Estos atributos son los que se conocen como selectores ID y selectores Class.
  • Definición de CLASS: Se entiende por un CLASS o CLASE, un atributo que podemos aplicar en HTML y CSS, para
    atribuir una característica a varias familias de etiquetas o elementos en nuestra web.
  • Para poder asignar un elemento CLASS en HTML, utilizaremos la estructura class = "nombre_asignado" y en CSS, se le representará
    como .nombre_asignado.
