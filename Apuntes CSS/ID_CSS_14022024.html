Definición de ID en CSS

En HTML y CSS, disponemos de una serie de atributos que nos permiten seleccionar ciertos elementos o etiquetas para asignarles
una serie de características concretas con respecto al resto de elementos que forman nuestra web.
Estos atributos son los que se conocen como selectores ID y selectores Class.
  • Definición de ID: Se entiende por un ID o Identificador Único, un atributo que podemos aplicar en HTML y CSS, para
    atribuir una característica única a un única familia de etiquetas o elementos en nuestra web.
  • Para poder asignar un elemento ID en HTML, utilizaremos la estructura id = "nombre_asignado" y en CSS, se le representará
    como #nombre_asignado.
