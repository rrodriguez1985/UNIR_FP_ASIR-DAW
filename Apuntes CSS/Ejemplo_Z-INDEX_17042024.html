Propiedad de Profundidad de Niveles (z-index)

La propiedad z-index, es un elemento que podemos usar en CSS para establecer profundidad en un elemento
sobre el resto.

Nos permite exactamente situar elementos (principalmente se utilizan en imagenes) una encima de la otra.

Su funcionamiento es el siguiente:
  • Para poder establecer una prounfidonda, sólo tendremos que indicar el nivel de profundidad del elemento.
  • Por ejemplo, si el nivel que le damos es mayor que 0, el elemento se posicionará por encima. (z-index=4)
  • Si el nivle que le damos es menor que 0, es negativo, el elemento se posicionará por debajo. (z-index=-2)
  • Si no le damos valor, o el valor que le damos es 0, el elemento se quedará tal y como aparece por defecto en nuestra web
  • Importante: Tener en cuenta, que la capa o nivel 0, es la capa desde la que partimos, es decir, es como se ha dicho
    la que viene por defecto.


Es muy importante indicar que los niveles z-indes así como las propiedades asociadas a estas como son: left, top, right y bottom,
NO FUNCIONAN con elementos que están posicionados con STATCI (Estático). Es decir, su posicionamiento debe ser distinto
a Static. Puede ser: relative, absolute, fixed, etc.
