Audio y Video en HTML5

En HTML5, que existen unas etiquetas específicas para poder insertar archivos tanto de audio como de video.
Las etiquetas que utilizaremos son: etiqueta audio de apertura y cierra para insertar audio.
La etiqueta video de apertura y cierre para insertar archivos de video.
Atributos comunes que podemos usar tanto para audio como para video
  • src: sirve para vincular el archivo de audio o vídeo a nuestra web
  • preload: es usato en e audio y vídeo para almacenar archivos de gran tamaño.
  • mediagroup: para realizar la agrupación de elementos multimedia.
  • autoplay: reproduce al inicio
  • loop: reproduce de manera infinita (bucle)
  • muted: silenciar el audio
  • controls: controles de nuestro reproductor multimedia

Atributos sólo para video
  • width y height
  • poster: insertar una imagen previa en el video
